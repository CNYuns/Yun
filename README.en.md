**3X-UI** is a web-based advanced open-source control panel designed for managing Xray-core servers. It provides a user-friendly interface for configuring and monitoring various VPN and proxy protocols.

> [!IMPORTANT]
> This project is for personal use and communication only. Do not use it for illegal purposes and do not use it in production environments.

As an enhanced version of the original X-UI project, 3X-UI provides better stability, broader protocol support, and additional features.

## Quick Start

```bash
bash <(curl -Ls https://gitee.com/YX-love/3x-ui/raw/main/install.sh)
```

For complete documentation, please refer to the [Project Wiki](https://gitee.com/YX-love/3x-ui/wikis).
